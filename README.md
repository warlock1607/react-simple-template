# React-simple-template

Minimal workflow template for react apps using webpack 4 + babel + express

## dependence

 - Node.js
 - npm

## Instalation

    git clone https://github.com/warlock1607/react-simple-template.git myapp
    cd myapp/
    npm i

## Usage

start dev-server:

    npm run dev
build you project:

    npm run build
start production build:

    npm start
