minimal workflow generator for react apps

## Instalation

    npm i react-simple-template -g

## Usage

generate template

    simple myapp
    
go to generated app folder

    cd myapp
install dependences

    npm i

start dev-server:

    npm run dev
build you project:

    npm run build
    
**note: comment out _module.hot.accept();_ in src/index.js before building**

start production build:

    npm start
